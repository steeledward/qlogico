Calculadora


Existe una calculadora que permite la manipulación de números en diferentes bases.

Se puede acceder de diferentes maneras.

Mediante el uso de la barra de herramientas, solo es necesario hacer click sobre el icono que muestra un gráfico como el que a continuación se muestra.



de igual manera es posible acceder mediante la barra de menus, seleccionando la opción calculadora del menú con el nombre Herramientas, por ultimo es posible acceder mediante la combinación de teclas ALT+K.

A continuación aparecerá un dialogo como el siguiente:



    Existen las siguientes operaciones disponibles:

Operación

Breve descripción

Conversión

    Permite convertir un numero a una base diferente.

+ Suma

    Realiza la suma aritmetica de dos números de diferente base.

- Resta

    Realiza la resta aritmetica de dos números de diferente base.

* Multiplicación

    Realiza el producto aritmetico de dos números de diferente base.

/ División

    Realiza la división aritmetica de dos números de diferente base.



Existe un campo en donde se activa el Numero en base R con su respectiva Base R



Existe un campo en donde se activa el Numero en base S con su respectiva Base S



Existe un campo en donde se muestra el Resultado deseado en la base 10



Asi mismo se puede seleccionar la Precisión con la cual se desea obtener los resultados

Conversión de bases


La calculadora incluida en Qlogico permite convertir un número a diferentes bases con una precision de hasta 10 digitos. La base del número deberá estar entre 2 y 36, la cantidad de digitos para expresar el número esta determinada por la capacidad del computador.

Ejemplo:

Se desea convertir el número 1010.01 el cual pertenece a la base 2 a una nueva base 10.

Es necesario abrir el dialogo de la Calculadora y llenar los campos como se muestra a continuación:



Como lo muestra el dialogo anterior hemos seleccionando una precisión de 2 digitos el Número en base R tiene el valor de 1010.01 y le hemos asignado el valor de su Base R a el valor 2, por ultimo solo es necesario activar la Base S con el valor de 10 ya que esta es la base a la cual queremos convertir el Número en base R, ahora solo resta hacer click en el boton que tiene la etiqueta Evaluar, a continuación aparecerá el dialogo modificado con la siguiente información:



en el cual se muestra el valor del Número en base S.

+ Suma


Para llevar a cabo la adición de dos números de diferente base solo es necesario llenar los campos Número en base R con su respectiva Base R y el Número en base S con su respectiva Base S y seleccionar el campo de operaciones la opción de + suma.

Ejemplo:

Deseamos conocer el valor de la suma de los numeros 1010 en base 2 y el número FF en base 16.

Solo es necesario llenar los campos con los valores conocidos y seleccionar la opción de + suma. A continuación tendremos el siguiente resultado:





el Resultado obtenido se presenta en base 10.
